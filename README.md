# ZRPC

## ZRPC介绍

ZRPC是一个高效、非侵入式的RPC调用框架，采用Netty NIO传输数据，Zookeeper管理服务，Protostuff、fastjson等多种序列化方式，并提供`Engine`接口供用户自定义序列化方式。
